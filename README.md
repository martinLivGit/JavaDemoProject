This was an exercise set by an investment bank, some time ago now, with time constraints, to demonstrate coding a solution in Java.

A constraint imposed was to minimise the use of third party software. I avoided using database software. The only significant thrid party software used was a simple JSON parser.

In addition, there was very little in terms of requirements which thus required the author to shape the solution and its design.

I have attempted to demonstrate a number of patterns including Singleton, Factory and loosely on Observer/Observable.

Finally, I have implemented a number of unit tests with test data files and the project is built with Maven.

At the time of writing, I included a word document but this has not been updated after recent changes.
